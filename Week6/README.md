# BIT3208 – Week 6: Database Integration and CRUD Operations
## Faraj Fruit Supplier and Vendor – Product Management System

### Pages
- `index.php` → CRUD Dashboard
- `create.php` → Add new product (INSERT)
- `read.php` → View all products (SELECT)
- `update.php` → Edit product (UPDATE)
- `delete.php` → Remove product (DELETE)

### Setup
1. Copy folder to `C:\xampp\htdocs\faraj\Week6\`
2. Make sure `faraj_db` is running in phpMyAdmin
3. Visit `http://localhost/faraj/Week6/`

### Database Used
- Database: `faraj_db`
- Table: `products`
